# OS Page Replacement Algorithm Visualizer

An interactive **Operating System mini-project** built using Python to visualize **swap in / swap out of memory pages** using different **Page Replacement Algorithms**.

This simulator helps students understand how **virtual memory management** works through **animations, themes, speed controls, and comparison graphs**.

---

# Features
- FIFO Page Replacement
- LRU Page Replacement
- Optimal Page Replacement
- Animated frame visualization
- Pause / Resume using keyboard
- Speed control
- Multiple visual themes
- Comparison graph for page faults
- Page fault counter
- Student-friendly learning simulator
